# 阿里云SAE

## 本地快速体验

通过该应用，您可以简单快速的使用SEA组件部署jar包/war包/zip包/镜像。

- 下载命令行工具：`npm install -g @serverless-devs/s`
- 初始化一个模版项目：`s init start-sae`
- 进入相应项目后部署项目：`s deploy`
